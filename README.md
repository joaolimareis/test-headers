# **Test Headers**

Este repositório contém um projeto simples em **Python (Flask)** criado para estudar e testar o funcionamento de **headers HTTP** na prática.

O objetivo é entender como requisições e respostas funcionam, explorando conceitos como:

* `Cache-Control`
* `Cookie`
* `Authorization`
* `X-Frame-Options`

As rotas desta aplicação permitem visualizar, enviar e manipular diferentes headers para aprendizado sobre:

* Como o cliente envia informações (Request Headers)
* Como o servidor responde (Response Headers)
* Como headers influenciam segurança, cache e comportamentos do navegador

Este projeto faz parte do meu estudo sobre **HTTP, HTTPS e segurança web**, servindo como ambiente de testes para compreender o fluxo completo entre **requisição → headers → resposta**.
